Faxin Zhou, Amirhossein Khalilian-Gourtani, Patricia Dugan, Andrew Michalak, Orrin Devinsky, Peter Rozman,Werner Doyle, Daniel Friedman, Adeen Flinker. Nature Communications, in press (2026). Pre-print: https://www.biorxiv.org/content/10.1101/2025.06.26.661755v1
