# audiovisual_frontal_organization
This repository contains the code and sample data for the following paper:

Frontal cortex organization supporting audiovisual processing during naturalistic viewing. Faxin Zhou, Amirhossein Khalilian-Gourtani, Patricia Dugan, Andrew Michalak, Orrin Devinsky, Peter Rozman,Werner Doyle, Daniel Friedman, Adeen Flinker. Nature Communications, in press (2026). Pre-print: https://www.biorxiv.org/content/10.1101/2025.06.26.661755v1

## Quick link to the related repository:
- mTRF toolbox: https://github.com/mickcrosse/mTRF-Toolbox
- Mithra (iEEG visualization): https://github.com/amirhkhalilian/Mithra
- pytorch: https://github.com/pytorch/pytorch
